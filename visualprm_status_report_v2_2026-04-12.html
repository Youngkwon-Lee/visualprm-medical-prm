Experiment Design

요청 반영: 3단계 실험

1) Baseline → 2) Baseline vs RAG → 3) Baseline vs RAG vs RAG_PRM

1

Baseline에서 실제 한 일

  • VQA-RAD/PathVQA 샘플의 질문+선택지(+이미지)를 Gemini에 입력
  • Gemini가 단계적 추론을 만든 뒤 최종 답 1개를 선택
  • 선택한 답을 gold와 비교해 정오답 계산
  • RAG/PRM 없이 모델 단독 성능을 기준선으로 사용
  • 결과: acc 약 0.50, 0번 쏠림
2

RAG 파이프라인 (쉬운 설명)

  • Route: 이 문제를 어떤 관점(일반/병리/영상)으로 볼지 결정
  • Retrieve: Qdrant에서 비슷한 사례/문서 근거 검색
  • Grounding: 찾은 근거를 붙여 판단 기준 고정
  • Reasoning: Gemini가 질문+선택지+근거로 단계적 판단
  • Final: 최종 답 1개 선택 (현재 baseline 대비 이득 미미)
3

RAG + PRM 파이프라인 (도식)

  • Input → RAG → BoN 후보 N개
  • → PRM score(candidate별)
  • → best candidate 선택 → final_answer
  • 모드: selection_mode=prm, bon_n
  • 현재: PRM 선택은 동작하나 후보가 0으로 붕괴
