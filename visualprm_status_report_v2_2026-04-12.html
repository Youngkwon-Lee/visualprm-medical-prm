Experiment Design

요청 반영: 3단계 실험

1) Baseline → 2) Baseline vs RAG → 3) Baseline vs RAG vs RAG_PRM

1

Baseline (현재 임시)

  • 현재: Gemini 단독으로 질문/선택지(+이미지) 입력
  • 모델이 단계적 추론 후 최종 답 1개 선택
  • gold와 비교해 정답률 계산
  • 추후: Qwen-VL 학습모델로 교체 예정
  • 이번 기준선: acc 약 0.50
2

Baseline vs RAG

  • RAG는 route → retrieve → grounding → reasoning 추가
  • 근거를 붙여 판단하도록 유도
  • 이번 결과에서는 baseline 대비 이득 미미
3

Baseline vs RAG vs RAG+PRM (유지)

  • RAG+PRM도 앞단 RAG(route/retrieve/grounding/reasoning)는 동일
  • 마지막 선택만 model 선택 → PRM 선택으로 교체
  • 현재: Gemini 임시 + PRM 경로 검증 단계
  • 목표: Qwen-VL 학습모델 + 팀원 real PRM 조합으로 전환
